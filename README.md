# CP2 Web Development

Projeto ecommerce de action figures desenvolvido para a disciplina de Web Development da FIAP.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- Git
- GitHub

## ⚡ Funcionalidades

- Cards de produtos com DOM
- Carrinho de compras
- Cálculo total com Reduce
- Desconto de 10%
- Responsividade
- Navegação entre páginas
